# lgloege.github.io
A webpage build with jekyll and SASS. This webpage highlights my graduate career.

# Notes
- may need to run `bundle update` or `gem install public_suffix --version 3.0.1` if get `Could not find public_suffix` error
# Serving the site
- `jekyll serve --watch` will serve the site and give a URL to access it
- `jekyll build` builds the site
- `rake publish` pushes the site to GitHub rep.
